# Opgave_2sem_musicdb

Opgave_2sem_musicdb is a fullstack app for your own personal music artist library.

## Installation

node.js is needed, the exe installs can be found at:
https://nodejs.org/en
Use npm to install the needed dependencies

```bash
npm install express
npm install cors
npm install uuid
```

## Usage

```bash
cd ./backend
npm start
```
